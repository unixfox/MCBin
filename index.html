Service Linux pour votre(vos) serveur(s) Minecraft !

Description

Ce projet a pour but de vous founir un script stable et facile d'utilisateur pour gérer votre(vos) serveur(s) Minecraft.

Fonctionnalités

Mono serveur

  • Démarrer / stopper votre serveur minecraft depuis une commande.
  • Peut être configuré.
  • Exécuter une commande sur son serveur sans ouvrir la console.
  • Ouvrir la console en une commande.

Multi serveurs (=Bungeecord) {à venir}

  • Démarrer / stopper vos serveurs et proxy en une fois ou démarrer / stopper un serveur en particulier.
  • Peut être configuré.
  • Exécuter une commande sur un serveur particulier sans ouvrir la console.
  • Ouvrir la console d'un serveur particulier en une commande.

Support

Vous pouvez ouvrir un ticket en cas de bug, anomalie, nouveauté sur le dépôt.

Néanmoins vous pouvez réagir au projet sur le forum de DevBukkit.

Participer au développement

Vous êtes développeur Bash / Shell ? Vous avez la possibilité d'améliorer mon code sur le dépôt ! N'hésitez pas, ça ne vous coûte rien !
